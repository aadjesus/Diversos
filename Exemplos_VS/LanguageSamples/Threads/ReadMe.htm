Threading Sample

[This topic is pre-release documentation and is subject to change in future releases. Blank topics are included as placeholders.]

This sample demonstrates the following threading topics. See Threading (C# Programming Guide) for more information.

  • Creating and executing a thread

  • Thread synchronization

  • Interaction between threads

  • Using a thread pool

Security Note This sample code is provided to illustrate a concept and should not be used in applications or Web sites, as it may not illustrate the safest coding practices.

To build and run the Threading samples within Visual Studio

  1. Open the solution (Threads.sln).

  2. In Solution Explorer, right-click the StopJoin project and click Set as StartUp Project.

  3. From the Debug menu, click Start Without Debugging.

  4. Repeat the preceding steps for MonitorSample, SimplePool, and Mutex.

To build and run the Threading samples from the Command Line

  1. Use the Change Directory command to change to the Threads directory.

  2. Type the following: 

    cd StopJoin
csc StopJoin.cs
StopJoin

  3. Type the following: 

    cd ..\MonitorSample
csc MonitorSample.cs
MonitorSample

  4. Type the following: 

    cd ..\SimplePool
csc SimplePool.cs
SimplePool

  5. Type the following: 

    cd ..\Mutex
csc Mutex.cs
Mutex
